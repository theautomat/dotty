# MANCER BLOG OUTLINE

This document serves as our living outline and planning document for the Mancer blog/documentation. We'll update this as we develop both the game and the blog.

## WRITING GUIDELINES

- Use a conversational, slightly humorous tone
- Include both technical details and in-universe lore
- Write for both casual players and technically-inclined readers
- Format code examples in code blocks
- Include references to sci-fi classics where appropriate
- Use green terminal-style headings and formatting in the final presentation
- Include visual cues that match the game's aesthetic
